Biscuit is a Rapid Application Development (RAD) framework for PHP 5 built on MVC programming standards. Some of its features include:

  • Rapid Development - Biscuit does most of the heavy lifting for you so you can focus all your time on deliverables.
  • Extendable Modular Design - Supports customizable modules and extensions so you can tailor your site to any need. Write your own, or customize all or part of any existing ones.
  • Themes - Apply a theme to the whole site or individual pages.
  • Security - Protects against most common website security threats.

You can begin setting up your new Biscuit site by logging into the System Administrator account.

The default username and password are admin / admin. You can change these after you login.

How do I Replace This Page?

To replace this page with your own content, simply create a file called index.html or index.php in the views folder in the root of your site. Fill it up with content and you will see it appear here.

Alternatively you can install one or more modules on the home page to deliver dynamic content.
